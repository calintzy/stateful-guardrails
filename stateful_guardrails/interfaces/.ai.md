# interfaces — AI 가이드

> 이 디렉토리를 수정하기 전에 반드시 이 파일을 읽는다.

## 이 디렉토리의 역할

사용자/운영자 진입점 레이어. CLI(`sgr`, typer 기반)가 주이며 선택적 FastAPI 서버도 여기에 위치한다. 가장 바깥 레이어로 모든 하위 레이어를 import할 수 있다.

## 허용되는 것

- `stateful_guardrails.core.*` import
- `stateful_guardrails.pipeline.*` import
- `stateful_guardrails.adapters.*` import
- `typer`, `fastapi`, `uvicorn` 등 UI·서버 라이브러리

## 금지되는 것

- 비즈니스 로직(판정·에스컬레이션 결정)을 interfaces에 직접 작성 금지 — pipeline/core의 책임
- DB 직접 접근 금지 — adapters를 통해서만

## 파일 구조

```
interfaces/
  __init__.py   ← 레이어 설명
  cli.py        ← typer app, sgr 커맨드 정의
  server.py     ← FastAPI 앱 (선택, Phase 3 이후)
```

## 코드 패턴

```python
# 올바른 예시 — typer 커맨드에서 pipeline 호출
import typer
from stateful_guardrails.adapters.llm import get_adapter

@app.command("ping-llm")
def ping_llm(provider: str = "ollama"):
    adapter = get_adapter(provider)
    typer.echo(adapter.complete("테스트"))
```

## 주의사항

- `sgr --version`은 `__version__`을 `stateful_guardrails` 패키지에서 가져온다.
- 커맨드 추가 시 `app.command()` 데코레이터만 사용하며, 새 typer sub-app은 `app.add_typer()`로 등록한다.
- ISC 번호가 붙은 커맨드(`sgr ping-llm`, `sgr scan` 등)의 출력 포맷은 변경 전 E장 ISC 기준과 대조한다.
